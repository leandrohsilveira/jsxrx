# Contributing

These documents cover internal implementation details of JsxRx. They are intended for developers who want to contribute to JsxRx or understand its internals. Reading them is not necessary for using JsxRx in applications.

1. [observables-internals.md](observables-internals.md) — VDOM reconciliation, ObservableDelegate, Input, State, and Batch Rendering internals
2. [components-internals.md](components-internals.md) — Props intake mechanics, lifecycle creation, VDOM flow
3. [jsx-internals.md](jsx-internals.md) — JSX transpilation pipeline, runtime functions, VDOM node types
4. [activity-aware-internals.md](activity-aware-internals.md) — ActivityAwareObservable, Suspense auto-detection, API client integration
